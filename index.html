Website LapaqPay membantu admin sebuah toko dalam mengelola transaksi penjualan secara cepat, rapi, dan efisien.

Cara Login

  1. Klik tombol Login yang terletak di pojok kanan atas menu navigasi utama.
  2. Masukkan Username pada kolom yang tersedia.
  3. Ketikkan Password Anda dengan teliti.
  4. Klik tombol Login berwarna biru di bawah kolom input
  5. Jika muncul notifikasi, tekan tombol OK untuk diarahkan ke halaman utama.

Cara Mencatat Transaksi

  1. Klik tombol transaksi yang terletak di bagian atas menu navigasi utama.
  2. Masukan data yang diminta di form transaksi sesuai keinginan.
  3. Jika sudah, tekan tombol simpan untuk menyimpan transaksi.
  4. Lihat daftar transaksi untuk melihat riwayat transaksi
  5. tekan tombol hapus jika ingin hapus transaksi

Cara Logout

  1. Klik tombol profil yang terletak di bagian atas menu navigasi utama.
  2. Tekan tombol logout yang ada di bagian bawah website
  3. Jika muncul keterangan, tekan OK
  4. Tunggu hingga proses logout selesai.
